filename( raw_filename )
==============================================================

Returns a String where all characters except: letters numbers underscores dashes are replaced with a dash. It also delets any non-alphanumeric characters at the end of the String.

==============================================================
html( raw_text )
=========================================

A better alternative than "Rack::Utils.escape_html". Escapes various characters (including ’&’, ’<’, ’>’, and both quotation mark types) to HTML decimal entities. Also escapes the characters from SWISS::HTML_ESCAPE_TABLE.

Text has to be UTF-8 before encoding, according to HTMLEntities gem. Therefore, all text is run through <Wash.plaintext> before encoding.

Takes out any periods and back slashes in a String. Single periods surround text are allowed on the last substring past the last slash because they are assumed to be filenames with extensions.

===================================================
plaintext( raw_str, *opts)
=========================================
  Returns: A string that is:
       * normalized to :KC
       * "\r\n" changed to "\n"
       * all control characters stripped except for "\n"
       * all control characters (including \n, \r) stripped at the beginning
         and end.

Options: 

   :tabs
   :spaces
