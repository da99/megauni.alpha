extract_emails( raw_str )

Returns an Array. First value is an Array with all the valid emails. It is an empty array if no valid emails are found. Second value is an Array with all values containing ’@’ but which are still invalid. It is an empty Array if no invalid emails found.

parse_into_hash( raw_body ) 
  Returns an Array with two values in the following format:
   [ "unparsable text string", {
       'SOME COLUMN' => 'some value'
       'ANOTHER COLUMN' => "All text \n could not \n be parsed \n in one line regardless of location in body"
   } ]

The Hash is empty if no parsable values are found.

parse_tags(tag_str_or_arr)

Parses tags from a string or an Array. Also parases multiple tags in a line within a multi-line string.

Raises: ArgumentError, if argument is not a string.

Returns: Array.
