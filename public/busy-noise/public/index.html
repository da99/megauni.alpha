The Complete Instruction Manual

Why use BusyNoise.com?
BusyNoise.com allows you to keep track of your data using just email. No installation or special software is necessary. All it takes is sending emails to your Busy Noise account.

* Create and manipulate your data from any device that can send and receive email.
* View your data as rows and columns from any web browser.
* Copy/paste your records into your favorite spreadsheet editor.
* Keep organized with tags.
 
Getting Started.
Once you create your account, you will receive a welcome email message. This
message contains a special email address ending in "@busynoise.com". This is your special email address that is assigned to you where you send your data to be recorded.

Use the instructions below so each email your write can be read and processed properly by the BusyNoise.com computer.
 
Creating a new record.
Add the # sign between the column name and value.

Put another # sign on a single line for the computer to ignore your
signature or legal notices your company puts at the bottom of your email.
======== SUBJECT LINE ========
NEW Oregon Race 2008

============ BODY ============
Miles # 2.5
Min # 10
Sec # 22
#
Create a shortcut for writing.
Text shortcuts are useful if you want to avoid typing a long phrase or name repeatedly.

Instead of writing:
Mr. Bob Johnson, Jr.
you could write:
bob2

When you retrieve your records, every instance of "bob2" will be replaced with "Mr. Bob Johnson, Jr."
======== SUBJECT LINE ========
Create BN-shortcut

============ BODY ============
bob2
Mr. Bob Johnson, Jr.
#
Getting records.
Send an email with the subject line starting with the word GET and
followed by the tags of the records you want to retrieve.
======== SUBJECT LINE ========
GET oregon race 2008
Getting records using filtering.
Use the GET command with tags at the end. Then add BUT NOT and the tags you want to exclude.

In the following example, all records tagged "vitamins" are retrieved, except for records tagged either "omega-3" or "biotin".
======== SUBJECT LINE ========
GET vitamins BUT NOT omega-3 biotin
Getting Untagged Records
The GET UNTAGGED command allows you to retrieve any records that have no tags.
======== SUBJECT LINE ========
Get Untagged
Getting records and specifying columns.
You can specify which columns you would like when retrieving your records.

Type COLS #. Then put each column on a separate line. Letter case is not important.
======== SUBJECT LINE ========
GET oregon race 2008

============ BODY ============
COLS #
Miles
Min
Get a list of all your tags.
To view a list of all the tags you have used, sorted alphabetically, use the GET MY TAGS command in the subject line.
======== SUBJECT LINE ========
Get My Tags
Updating record fields.
Updating records works the same as creating records. The main difference is in the subject line.

Use the UPDATE word followed by the Busy ID of the record you want to update.
======== SUBJECT LINE ========
UPDATE 51234

============ BODY ============
MILES # 5000
NOTE #
Arrived later than usual.
#
Changing the spelling of a tag for all records.
With the CHANGE TAG command, you can fix or change the spelling of a tag used by multiple records. This is useful if you have been using different spelling forms of the same tag (e.g. race, races) and would like to standardize them all to one spelling.
======== SUBJECT LINE ========
Change Tag race

============ BODY ============
races
#
Adding tags to a record.
You can add tags to an existing record by using the UPDATE command in the subject and ADD TAGS in the body.
======== SUBJECT LINE ========
UPDATE 51234

============ BODY ============
ADD TAGS #
rainy morning
#
Removing tags from a record.
Just like adding a tag, but instead write REMOVE TAGS.
======== SUBJECT LINE ========
UPDATE 51234

============ BODY ============
REMOVE TAGS #
rainy morning
#
Appending a record field.
Appending recording values works the same way as UPDATE. The only difference is that instead of overwriting the entire field value in a record, the text is combined with the old value.
======== SUBJECT LINE ========
APPEND 51234

============ BODY ============
NOTE #
It was a cold and rainy morning.
#
Deleting a record.
You can delete an entire record (row). There is no undo. However, you do receive a copy of the record you deleted sent to your email address.
======== SUBJECT LINE ========
DELETE 51234
Get a copy of this manual as an email.
======== SUBJECT LINE ========
GET My Manual
