Public Class methods

attach_owner_to_letter( letter )

Accepts a letter (Hash) from SurferMail. If either address is not approved either as a primary address or a email alias, then a letter is mailed to Member telling them to confirm address as an alias. Exception ‘EmailAlias::ERRORS::PendingAlias’ is thrown.

Retrieves the Member belonging to a letter. Creates email alias if :sender and :from does not match. Calls :mail_box_owner_or_not! to validate mail box address. Adds :owner_mail_box to letter.

Raises ERRORS::NoRecordFound if member can not be located from either address.
