Module Sanitize
In: vendor/html5_sanitize.rb

Introduction

This module provides sanitization of XHTML+MathML+SVG and of inline style attributes. Its genesis is described here.

Uses the HTML5lib parser, so that the parsing behaviour should resemble that of browsers. 

 sanitize_xhtml() is a case-sensitive sanitizer, suitable for XHTML
 Sanitize.html() is a case-insensitive sanitizer suitable for HTML
 sanitize_rexml() sanitizes a REXML tree, returning a string

Sanitize.html( string, {:encoding => 'iso-8859-1', :to_tree => true} ) → REXML::Document

Sanitize a string, parsed using HTML parsing rules.

Unless otherwise specified, the string is assumed to be utf-8 encoded. By default, the output is a string. But, optionally, you can return a REXML tree.

The string returned is utf-8 encoded. If you want, you can use iconv to convert it to some other encoding. (REXML trees are always utf-8 encoded.)
